1. WWW

World Wide Web (WWW) is Collection of HTML(HyperText Markup language) documents. While almost all types of files can exist on the web, any type of file can be linked to HTML documents that are the basis for everything on the web.

2. Major Pieces

Your computer is running a Web browser, the Internet, and servers. Your browser makes requests via the Internet to servers. These requests are using a protocol called http and the servers respond with files that your browser displays. Some examples of browsers are Internet Explorer, Chrome, Firefox.

  • HTTP – the main protocol of the web
  • Servers – computers that host the files that make up the web
  • Internet – the world’s largest computer network
  • Browser – a program that runs on your computer to display files found on the web

3. HTML(HyperText Markup Language) is made up of:

  • Text content (What you see)
  • Markup (what it looks like)
  • References to other documents, e.g. images, videos
  • Links to others

4. Intro to HTML Tags

  • Tags, once opened, need to be closed!
  • HTML Attributes <Tag Attr="value">contents</tag>"

5. Types of HTML tags

  • Bold tag - <b> </b>
  • Italics tag - <em> </em>
  • Anchor tag <a> - makes a link, <a href=”www.reddit.com” sung </a>
  • Image tag - <img src="URL" alt="text">
    Image tag is a void tag as there is no contents and therefore do not require a closing tag.
    Src = source alt = alternative name
  • Break tag - <br>
    Break tag is a void tag and do not need a closing tag. Can be used multiple times to give multiple line breaks. It ends the line and it is a lineline element (stays within the line), although it creates multiple lines as it just creates white space within the line pushing the rest to the next line.
  • Paragraph tag - <p> contents </p>
    It makes the invisible box (block element) around the contents.
  • "\n" that Steve writes is how programmers represent a "newline" character in common coding languages such as C, Java, Python, etc.
  • Span/Div - They are same but Span inline element Div is a block element.Their only function is to contain contents.
    - The <span> tag is used to group inline-elements in a document.
    - The <span> tag provides no visual change by itself.
    - The <span> tag provides a way to add a hook to a part of a text or a part of a document.
